
# Transfer Learning para Classificação de Imagens

## Descrição
Projeto desenvolvido para o desafio da DIO demonstrando Transfer Learning com TensorFlow.

O modelo MobileNetV2 pré-treinado é utilizado como base para treinar um classificador de duas classes.

## Tecnologias
- Python
- TensorFlow
- Keras
- Google Colab

## Estrutura do Projeto

dataset/ -> imagens para treinamento  
notebook/ -> notebook principal  
images/ -> capturas de tela dos resultados  

## Como Executar

1. Abra o notebook no Google Colab
2. Faça upload do dataset para a pasta dataset
3. Execute todas as células
4. Observe os resultados de treinamento

## Dataset

Estrutura esperada:

dataset/
    class1/
    class2/

Cada pasta deve conter imagens da respectiva classe.
